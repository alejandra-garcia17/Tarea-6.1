Invertir una cadena

Respuesta:

Explicacion del codigo

- Primero, tomamos la cadena de texto que ingresamos en el campo de texto de la página web.

- Segundo, "const cadena = document.getElementById('cadena').value;:" Obtiene el valor del campo de texto. Y "let cadenaInvertida = "";:" Inicializa una cadena vacía para almacenar la cadena invertida.

- Tercero, "for (let i = cadena.length - 1; i >= 0; i--) { ... }:" Itera sobre cada carácter de la cadena, empezando desde el último. Y "cadenaInvertida += cadena[i];:" Agrega cada carácter a "cadenaInvertida" en orden inverso.

- Cuarto, "resultadoDiv.innerHTML = La cadena invertida es: ${cadenaInvertida}:" Muestra la cadena invertida en el div de resultados.
