Convertir mayúsculas a minúsculas y viceversa

Respuesta:

Explicacion del codigo

- Primero, tomamos la cadena de texto que ingresamos en el campo de texto de la página web.

- Segundo, "let nuevaCadena = "";": Inicializa una cadena vacía para almacenar la nueva cadena con las letras convertidas.

- Tercero, "for (let i = 0; i < cadena.length; i++) { ... }": Itera sobre cada carácter de la cadena. Y "let char = cadena[i];": Obtiene el carácter actual.

- Cuarto, "if (char >= 'a' && char <= 'z') { nuevaCadena += String.fromCharCode(char.charCodeAt(0) - 32); }": Si el carácter es una letra minúscula, convierte a mayúscula. O "else if (char >= 'A' && char <= 'Z') { nuevaCadena += String.fromCharCode(char.charCodeAt(0) + 32); }": Si el carácter es una letra mayúscula, convierte a minúscula. Y "else { nuevaCadena += char; }": Si el carácter no es una letra, lo agrega sin cambios.

- Quinto, "resultadoDiv.innerHTML = La cadena convertida es: ${nuevaCadena};": Muestra la nueva cadena con las letras convertidas en el div de resultados.
