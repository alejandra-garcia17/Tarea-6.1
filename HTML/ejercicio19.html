Contar las vocales en una cadena

Respuesta:

Explicacion del codigo

- Primero, tomamos la cadena de texto que ingresamos en el campo de texto de la página web.

- Segundo, "const cadena = document.getElementById('cadena').value.toLowerCase();": Obtiene el valor del campo de texto y lo convierte a minúsculas.

- Tercero, "for (let i = 0; i < cadena.length; i++) { ... }:" Itera sobre cada carácter de la cadena. La condicional "if (vocales.includes(cadena[i])) { contador++; }:" Verifica si el carácter actual es una vocal. Si es así, incrementa el contador.

- Cuarto, "resultadoDiv.innerHTML = La cadena tiene ${contador} vocales.;:" Muestra el número de vocales en el div de resultados.
