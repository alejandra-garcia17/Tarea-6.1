- Primero, tomamos el número que ingresamos en el campo de texto de la página web.

- Segundo, el "input.split(','):" sirve para dividir el valor ingresado por el usuario en un array de cadenas, usando la coma como separador. El ".map(num => parseInt(num.trim())):" Recorre cada elemento del array, eliminando espacios en blanco y convirtiéndolo en un número entero

- Tercero, "for (let i = numeros.length - 1; i >= 0; i--) { ... }:" Itera sobre cada número en el array empezando desde el último elemento. Y la condicional "if (numeros[i] % 2 === 0) { ... }:" Verifica si el número actual es par
