División de dos números por restas sucesivas

Respuesta:

Explicacion del codigo

- Primero, tomamos el número que ingresamos en el campo de texto de la página web.

- Segundo, se inicializa una variable "cociente" con el valor 0. Esta variable contará cuántas veces se puede restar el divisor del dividendo.

- Tercero, "while (dividendo >= divisor) { ... }" aquí comienza un ciclo while que continuará mientras el valor del dividendo sea mayor o igual al valor del divisor. Mientras se incrementa en 1 el valor de "cociente" por cada vez que se realiza la resta. Esto cuenta cuántas veces el divisor puede ser restado del dividendo.

- Y por ultimo, una vez que el ciclo while termine, se asigna al contenido de resultado el mensaje que muestra el valor del cociente obtenido.
