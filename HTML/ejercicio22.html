Reemplazar espacios con guiones

Respuesta:

Explicacion del codigo

- Primero, tomamos la cadena de texto que ingresamos en el campo de texto de la página web.

- Segundo, "let cadenaModificada = "";": Inicializa una cadena vacía para almacenar la cadena modificada.

- Tercero, "for (let i = 0; i < cadena.length; i++) { ... }": Itera sobre cada carácter de la cadena.

- Cuarto, "if (cadena[i] === " ") { cadenaModificada += "-"; } else { cadenaModificada += cadena[i]; }": Reemplaza cada espacio con un guion y mantiene los demás caracteres sin cambios.

- Quinto, "resultadoDiv.innerHTML = La cadena modificada es: ${cadenaModificada};": Muestra la cadena modificada en el div de resultados.
